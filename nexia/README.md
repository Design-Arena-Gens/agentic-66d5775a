Marketing site for Nexia, an AI agency delivering automation, machine learning, web, voice, and growth solutions for businesses of every size.

### Tech stack
- Next.js App Router with TypeScript
- Tailwind CSS v4 for theming
- Server-rendered landing page with contact form via Formspree

### Local development
```bash
npm install
npm run dev
```
Visit `http://localhost:3000` to view the site.

### Production build
```bash
npm run build
npm run start
```

Deploy-ready for Vercel (`npm run build` succeeds and outputs `.next/`).
